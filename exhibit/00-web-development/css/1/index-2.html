Flex Children Do Not "Fill Space" exhibit/00-web-development/css/1/index-2.html
A flex child can only stretch inside a parent that owns height.
This is the hidden thing that caused your navbar system to change behavior.
❌ Broken Mental Model
.parent { display: flex; } .child { flex: 1; height: 100%; }
Sidebar
No height context
✅ Correct Height Ownership
.parent { height: 400px; }
Sidebar
