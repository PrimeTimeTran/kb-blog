Flex Children Do Not "Fill Space"

A flex child can only stretch inside a parent that owns height.
This is the hidden thing that caused your navbar system to change behavior.

❌ Broken Mental Model

.parent { display: flex; } .child { flex: 1; height: 100%; }

The child looks like it should fill height.

But 100% of WHAT?

Sidebar
This does NOT truly fill height.

Parent never defined one.

✅ Correct Height Ownership

.parent { display: flex; height: 400px; } .child { flex: 1; height: 100%; }

Parent now owns vertical space.

Child finally knows what "100%" means.

Sidebar
This NOW fills height correctly.
